# HLD 테스트 케이스 문서 v0.1.0

## 목적
- 상위 설계(HLD) 시나리오를 재현 가능한 케이스로 검증한다.

## 범위
- 인증, 주문, 외부연계, 스케줄 실행의 상위 흐름.

## 테스트 케이스 목록
| ID | 시나리오 | 사전조건 | 절차 | 기대결과 |
|---|---|---|---|---|
| HLD-TC-001 | 인증 성공 | 유효 계정 | 로그인 요청 | 토큰 발급 및 성공 응답 |
| HLD-TC-002 | 인증 실패 | 비활성 계정 | 로그인 요청 | 인증 실패 코드 반환 |
| HLD-TC-003 | 주문 생성 | 인증 완료 | 주문 생성 호출 | 주문 상태 저장 |
| HLD-TC-004 | 배치 실행 | 스케줄러 활성 | 지정 시각 대기 | 배치 성공 로그 기록 |

## 수용 기준
- 문서 절차만으로 2회 이상 동일 결과를 재현할 수 있어야 한다.
- 실패 시 원인을 로그로 식별할 수 있어야 한다.
